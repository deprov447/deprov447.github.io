Median

The project consists 2 parts :

1) Median API: An independent GraphQL based open sourced blogging API designed to provide front end apps with all basic blog features. It is a platform agnostic API therefore any front end framework on any device can be be integrated with it as long as it can query GraphQL.

2) Median Web Client : A React.js based single page front end application that connects with Median API to provide a clean blogging interface. The web client tries to as similar as possible to medium.com but without any paywalls and user data logging.

Fake Feed

FakeFeed is a Kirigami based application that replaces the original camera feed with a virtual video devices. Normal (v4l2) applications will read these devices as if they were ordinary video devices, but the video will not be read from e.g. a capture card but instead it is generated by this application.
